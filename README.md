# Portfolio Website

Personal portfolio built with Vite + Next.js.

## Project Structure

- `src/` - main application source
  - `app/` - top-level app routes
  - `components/` - UI and portfolio components
  - `pages/` - page components
- `public/` - static assets (favicon, images)
- `index.html` - entry HTML
- `package.json` - dependencies and scripts

## Favicon

- `public/favicon.svg` is the site icon.

## Local Development

```bash
pnpm install
pnpm run dev
```

## Build

```bash
pnpm run build
```
